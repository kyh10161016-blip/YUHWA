# CARE LINK · 스마트 병원 케어 앱

CARE LINK 스마트 커뮤니케이션 디바이스와 연동되는 **모바일 전용 병원 케어 앱 UI** 프로토타입입니다.
간호사·의료진이 환자 요청, 긴급 상황, 위험 감지 알림, 인수인계를 핸드폰 화면에서 즉시 확인·처리할 수 있도록 설계했습니다.

## 주요 기능
- **홈 대시보드** — 담당 환자 수, 긴급 알림, 대기 요청, 인수인계 요약, 디바이스 연결 상태
- **실시간 협업 보드** — 환자별 상태 카드, 우선순위, 대기/진행 중/완료 구분
- **환자 요청 관리** — 응급/일반/안내 태그, 담당자 배정·처리 완료
- **인수인계 자동 정리** — 투약·검사·주의사항·미처리 업무 AI 요약 카드
- **환자 설명 가이드** — 검사 전·투약·퇴원 안내 문구 제공
- **위험 감지 알림** — 낙상·이상 생체신호·응급 호출·장시간 미처리, 경고 레벨별 카드
- **긴급 상황 지원** — 큰 알림 카드, 환자 위치, 즉시 호출·의료진 공유·기록
- **하단 탭 바** — 홈 / 환자 / 요청 / 알림 / 설정

## 기술
- 순수 **HTML / CSS / JavaScript** (React·Vue·Tailwind·외부 라이브러리 의존성 없음)
- 390 × 844 세로형 모바일 프레임 기준
- 모든 UI는 더미 데이터로 동작

## 실행
```bash
# 의존성 없는 Node 정적 서버
node server.js
# → http://localhost:8000
```
또는 `index.html`을 브라우저에서 직접 열어도 됩니다.

## 배포
GitHub Pages 설정에서 `main` 브랜치 / `/ (root)`를 지정하면 바로 배포됩니다.
